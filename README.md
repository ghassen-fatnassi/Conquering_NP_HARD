# Conquering_NP_HARD
in this repo , i implement different solutions to NP hard problems , i started with TSP.

## Implemented Algorithms
- **Nearest Neighbor**
- **Greedy Algorithm with Disjoint Set Union (DSU)**
- **Christofides Algorithm**
- **Simulated Annealing**
- **Genetic Algorithm**
- **Ant Colony Optimization**
